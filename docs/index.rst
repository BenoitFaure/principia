Principia
=========

Principia is an interface for generating a **Constitution** used in Supervised
Learning and Reinforcement Learning from Human Feedback (RLHF). It also generates
the datasets needed for both training phases. The approach is based on Anthropic's
`Constitutional AI <https://arxiv.org/abs/2212.06950>`_ paper.
